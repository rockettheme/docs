FP RokAjaxSearch
-----

![](assets/demo_1.jpeg)

:   1. **RokAjaxSearch** [40%, 81%, se]

The Site Search area of the front page is a **mod_rokajaxsearch** module that allows visitors to search your site using the powerful RokAjaxSearch tool.

### Details

![](assets/demo_1a.jpeg)

| Option      | Setting            |
| :---------- | :----------        |
| Title       | `FP RokAjaxSearch` |
| Show Title  | Hide               |
| Position    | header-d           |
| Status      | Published          |
| Access      | Public             |
