Gantry Extras
-----

![][demo]

We used a **RokSprocket** module with the **Lists** layout to make up the content in the **sidebar-b** position of the front page. You will find the settings used in our demo below.

### Details

![][demo2]

| Option           | Setting       |  
| :--------------- | :------------ |  
| Title            | Gantry Extras |  
| Show Title       | Show          |  
| Access           | Public        |  
| Position         | sidebar-b     |  
| Status           | Published     |  
| Content Provider | Joomla        |  
| Type             | Lists         |
